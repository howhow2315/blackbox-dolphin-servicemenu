## blackbox-dolphin-servicemenu
Blackbox Dolphin extension for "Run in Blackbox" and "Open Blackbox here" Service Menu functions

![Run Example](https://raw.githubusercontent.com/howhow2315/blackbox-dolphin-servicemenu/main/examples/run_example.png)
![Open Example](https://raw.githubusercontent.com/howhow2315/blackbox-dolphin-servicemenu/main/examples/open_example.png)

---
#### Install/Uninstall

```sh
curl -fL https://github.com/howhow2315/blackbox-dolphin-servicemenu/raw/main/install.sh | sh # Install
```
```sh
curl -fL https://github.com/howhow2315/blackbox-dolphin-servicemenu/raw/main/uninstall.sh | sh # Uninstall
```

Installs extensions at
```
~/.local/share/kio/servicemenus/
```
